# 生成带绝对路径的目录树 Generte a Menu Tree with absolute paths appended

## 使用环境 Environment
- OS: Windows10
- Python Version：3.7.4
- Modules needed: codecs, chardet

## 使用方法 How to use
` python MenuTree-withAbsolutePathGenerator.py [directory]`
